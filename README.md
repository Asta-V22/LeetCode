## 🚀 Getting Started

### Prerequisites
- C++11 or higher for compiling C++ solutions
- Java JDK 8 or higher for running Java solutions
- A LeetCode account (optional, for reference)

### How to Use

1. **Browse solutions** by difficulty level or topic
2. **Review the code** to understand different approaches
3. **Compile and test** solutions locally
4. **Compare your approach** with the implementations here

### Compiling C++ Solutions

```bash
g++ -std=c++11 solution.cpp -o solution
./solution
```

### Running Java Solutions

```bash
javac Solution.java
java Solution
```
